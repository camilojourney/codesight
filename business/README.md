# CodeSight Business

Consulting business selling AI-powered document search to mid-market companies.

## Quick Reference

| | |
|--|--|
| **Product** | CodeSight — hybrid AI search over document collections |
| **Target** | Companies with 50-500 employees, scattered docs, no good search |
| **Pilot** | $7,500-10,000 / 2 weeks / 1 project / money-back guarantee |
| **Scale** | $3,000-5,000 per additional project |
| **Maintenance** | $1,000-2,000/month |
| **Differentiator** | Hybrid BM25+vector+RRF. 100% local search. Open source. |

## Files

| File | When to use |
|------|-------------|
| `proposals/templates/pitch.md` | Study before any client meeting |
| `proposals/templates/proposal.md` | Customize and send to qualified leads |
| `proposals/templates/one-pager.md` | Leave-behind PDF after meetings |
| `proposals/templates/deck.md` | Slide-by-slide script for presentations |
| `pipeline.md` | Track active leads and closed deals |
| `playbooks/sales-process.md` | The sales process (stages, discovery, pricing) |
| `playbooks/client-onboarding.md` | What happens after contract signed |

## Workflow

1. Lead comes in -> add to `pipeline.md`
2. Discovery call -> use questions from `pitch.md`
3. Qualified -> customize `proposal.md` + `one-pager.md`, save to `proposals/clients/<name>/`
4. Meeting/demo -> study `pitch.md`, use `deck.md` for slides
5. Closed -> move in `pipeline.md`, start `playbooks/client-onboarding.md`
